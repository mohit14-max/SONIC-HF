# SONIC

SONIC is a Gradio-based AI workspace app.

It now includes both chat and image generation modes in the same UI. Set `NVIDIA_API_KEY` in `.env.local` to enable image generation.

For local hosting, LAN access, public share links, and deployment notes, see [README_HOSTING.md](./README_HOSTING.md).
